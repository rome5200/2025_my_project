# 2025 My IoT Project  
> _2025.11.20 ~ 2025.11.21

## 🚀 프로젝트 소개
이 레포지토리는 **KY-AISW OpenSource_IoT를 활용한 GitHub 사물인터넷 프로그래밍 캠프**에 참여한 결과물입니다.  

## 🎯 주요 목표

### 🧩 Open Source Environment  
일상 속에서 자연스럽게 활용되고 있는 **오픈소스(Open Source)**의 개념과 역할을 이해합니다.  
우리의 tiny project를 위해 몇 가지 핵심 오픈소스를 직접 **셋업하고 연결해보며**,  
오픈소스가 주는 **편리함·확장성·자유도**를 체감하는 데 목적이 있습니다.

### 🗂️ Git / GitHub  
오픈소스 프로젝트 관리의 핵심 도구인 **Git**과 협업 플랫폼 **GitHub**를 활용합니다.  
수많은 개발자들이 Git/GitHub을 통해 자신의 아이디어를 효율적으로 기록하고 공유하듯,  
본 프로젝트에서도 **버전 관리·브랜치 전략·이슈 관리·GitHub 꾸미기** 등을 실습하여  
여러분만의 **포트폴리오로 성장할 수 있는 레포지토리**를 만들어가는 것을 목표로 합니다.

### 📡 Internet of Things (IoT)  
IoT는 우리 주변에 존재하는 또 하나의 “컴퓨터”입니다.  
PC와는 다른 방식으로 동작하는 IoT 장비를 제어하며,  
**컴퓨터의 원리·센서 데이터 흐름·하드웨어를 움직이는 소프트웨어의 힘**을 체험합니다.  
이를 통해 주변 환경을 자동화하고 데이터로 제어하는 IoT의 본질을 이해하는 것을 목표로 합니다.
